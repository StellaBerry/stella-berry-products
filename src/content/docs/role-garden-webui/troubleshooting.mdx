---
title: トラブル対応
description: 起動しない・Ollama関連・生成が遅い・スマホから開けない、といった場合のよくある対処。
---

import { Card, CardGrid } from '@astrojs/starlight/components';

<CardGrid>
  <Card title="起動しない" icon="warning">
    - zip を展開してから実行してください。
    - `Role Garden WebUI Launcher.exe`、または起動 bat を実行してください。
    - セキュリティソフトが Python・bat・exe を止めていないか確認してください。
  </Card>
  <Card title="Ollama 関連" icon="warning">
    - Ollama が未導入の場合は公式サイトから導入してください。
    - 既存の Ollama があれば再利用します。
    - モデル取得には時間とストレージ容量が必要です。
  </Card>
  <Card title="生成が遅い" icon="warning">
    - 重いモデルを選んでいる可能性があります。
    - まず軽量モデルで動作確認してください。
    - 速度は PC 性能・モデル・Ollama の状態に左右されます。
  </Card>
  <Card title="スマホから開けない" icon="warning">
    - PC とスマホが同じ Wi-Fi か確認してください。
    - Windows ファイアウォールで Python、またはポート 7880 が許可されているか確認してください。
    - ルーターの端末分離、VPN、仮想ネットワークの影響も確認してください。
  </Card>
</CardGrid>

## バックアップ

通常のデータ保存先は `Documents\RoleGardenWebUI\` です。アップデート・再インストール・アンインストール・データ初期化・手動編集の前には、必要に応じてバックアップしてください。
