Windows 64bit 向けです。難しい設定は要りません。**zip を展開して、ランチャーをダブルクリックするだけ**で始められます。

## かんたんセットアップ（推奨）

<Steps>

1. 配布 zip を、好きな場所に展開します。
2. **`Role Garden WebUI Launcher.exe`** をダブルクリックします。
3. 初回起動時、ランチャーが自動で次を行います。
   - お使いの **PCスペック（GPU / VRAM）を検出** し、**推奨モデルセット** を提示します。
   - **データとモデルの保存先** を選べます。
   - Ollama の用意を確認し、**モデルを進捗バー付きでダウンロード** します。
4. 完了すると、そのまま **自動で起動** し、ブラウザで WebUI が開きます。

</Steps>

あなたの操作は、基本的に「**ダブルクリック**」と「**モデルの確認**」だけです。

:::tip[Python のインストールは不要]
専用の Python ランタイムを `runtime\python` に同梱しています。お使いのシステムの Python は変更しません（アプリは標準ライブラリのみで動作します）。
:::

## 推奨モデルとスペックの目安

ランチャーは、お使いの GPU に合うプロファイルを **「推奨」** として表示します。VRAM に対して重すぎるセットを選ぼうとすると、**無駄な大容量ダウンロードを防ぐための警告** を出します。

| プロファイル | 取得するモデル | 目安 |
| --- | --- | --- |
| `low`  | gemma4:e2b, gemma4:e4b | 小さめの GPU や、CPU 中心の環境向け |
| `mid`  | ＋ qwen3:14b | VRAM 11GB 以上向け |
| `high` | ＋ gemma4:26b | VRAM 20GB 以上向け |

:::caution
VRAM に収まらない重いモデルは、生成に数分かかったり、環境によっては固まったように見えることがあります。迷ったら、まず **推奨（多くの環境では `low`）** から始めてください。
:::

## Ollama とモデルについて

- すでに **Ollama** が入っている場合は再利用します（自動更新はしません）。
- すでに取得済みのモデルはスキップし、**足りないモデルだけ** ダウンロードします。
- モデルデータは大きいため、初回のダウンロードには時間とディスク容量が必要です。

## データとモデルの保存先

- 初回のランチャー画面で選べます。
- 既定は、データが `Documents\RoleGardenWebUI`、モデルが Ollama の既定フォルダです。
- データをアプリのフォルダの外に置くことで、新しい zip でアプリを入れ替えても安全です。

## 起動・停止

- **起動**：`Role Garden WebUI Launcher.exe`（または `start_role_garden_webui.bat`）。
- **停止**：ランチャーの停止ボタン（または `stop_role_garden_webui.bat`）。

## クラシック（メニュー）セットアップ（任意・フォールバック）

ランチャーが使えない場合の互換手段として、従来のメニュー式セットアップも残しています。`install_role_garden_webui.bat` をダブルクリックすると、ランチャーが見つからないときはメニューにフォールバックします。

:::caution
LAN公開は家庭内LAN向けの機能です。インターネットへ直接公開しないでください。
:::
