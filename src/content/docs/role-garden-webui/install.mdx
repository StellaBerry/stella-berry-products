Windows 64bit 向けです。難しい設定は要りません。**zip を展開して、ランチャーをダブルクリックするだけ**で始められます。

## セットアップの手順

<Steps>

1. 配布 zip を、好きな場所に展開します。
2. **`Role Garden WebUI Launcher.exe`** をダブルクリックします。
3. あとはランチャーが、お使いのPCに合ったモデルを用意し、そのまま起動します。

</Steps>

初回は、PCスペックの確認・保存先の選択・モデルのダウンロード（進捗バー表示）まで、画面の案内どおりに進めるだけです。あなたの操作は、基本的に「ダブルクリック」と「モデルの確認」だけです。

## 必要なもの

- Windows 64bit のPC
- 初回セットアップ時のインターネット接続

## セットアップで自動的に用意されるもの

- Ollama（お使いのPCに無い場合）
- お使いのPCに合った AIモデル

Python などの追加インストールは要りません。

## モデルの選び方

ランチャーが、お使いの GPU に合ったモデルを **自動でおすすめ** します。迷ったら、おすすめのまま進めれば大丈夫です。重すぎるモデルを選ぼうとしたときは、警告が出ます。

| おすすめ | モデル | 目安 |
| --- | --- | --- |
| `low`  | gemma4:e2b, gemma4:e4b | 小さめの GPU / CPU 中心 |
| `mid`  | ＋ qwen3:14b | VRAM 11GB 以上 |
| `high` | ＋ gemma4:26b | VRAM 20GB 以上 |

:::caution
重すぎるモデルは、生成にとても時間がかかったり、固まったように見えることがあります。迷ったら、まずおすすめ（多くは `low`）から始めてください。
:::

## 保存先

データとモデルの保存先は、初回の画面で選べます（既定のままでも大丈夫です）。

## 起動・停止

- **起動**：`Role Garden WebUI Launcher.exe`
- **停止**：ランチャーの停止ボタン

:::note
うまくいかないときのために、従来のメニュー式セットアップ（`install_role_garden_webui.bat`）も残しています。
:::

:::caution
LAN公開は家庭内LAN向けの機能です。インターネットへ直接公開しないでください。
:::
