最初に迷ったら、**軽量構成**から始めてください。重いモデルを無理に使うと、PCが非常に重くなったり、生成に時間がかかったり、環境によってはフリーズすることがあります。

<Steps>

1. 配布 zip を展開します。
2. まず軽量構成で試す場合は `install_low_spec.bat` を実行します。
3. メニュー式で選びたい場合は `install_role_garden_webui.bat` を実行します。
4. インストール後、`Role Garden WebUI Launcher.exe` を開きます。
5. ランチャーが起動し、しばらくするとブラウザで WebUI が開きます。

</Steps>

## 構成メニュー

| メニュー | 用途 |
| --- | --- |
| `low` | まず動作確認したい方向けの軽量構成。迷ったらここから始めてください。 |
| `mid` | 軽量構成に加えて中規模モデルを取得。軽さと品質のバランスを試したい方向け。 |
| `high` | 高スペックPC向け。容量・RAM・VRAM に余裕がある場合のみ選んでください。 |
| `env` | Python・WebUI専用環境・Ollama だけを準備します（モデルは取得しません）。 |

## インストールされるもの

- Python 3.11 64bit、または既存の Python 3.10 以上 64bit
- このフォルダ内の WebUI 専用 Python 環境
- Ollama
- 選択した構成に応じたローカルLLMモデル

既に使える Python・Ollama・取得済みモデルがある場合は、できるだけ既存のものを再利用します。
