title: プロフィール

import { Image } from 'astro:assets';

自分が操作する、SNS アカウントのような「自キャラ」の画面です。

<Image src={shot} alt="プロフィール画面" class="rg-screenshot" />

- **名前 / 表示名** を設定できます。
- **プロフィール文** で人物像を書けます。
- **アイコン画像 / ヘッダー画像** を設定できます（画像は表示用で、AI の入力には使われません）。
- 複数の自キャラを作り、切り替えて遊べます。
