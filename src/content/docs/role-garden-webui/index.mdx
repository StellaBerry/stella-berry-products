見に来てくださって、ありがとうございます。
**Role Garden WebUI** は、自分のPCの中だけで動く「ローカルAI」で、キャラクターとの会話・SNS風のタイムライン・DMロールプレイを楽しむための Windows 向けアプリです。

自分側のキャラクターをSNSアカウントのように作り、そのキャラクターに紐づく相手キャラクターたちと、タイムラインやDMでやり取りできます。入力も生成結果もPCの外に出ないので、気兼ねなく物語を育てられます。

<CardGrid>
  <Card title="できること" icon="pencil">
    - 世界観・自キャラ・相手キャラの作成
    - SNS風タイムライン、リプライ、DM
    - ロールプレイ向けの返信生成
    - 文体・口調の分析
    - テーマ、カラーパレット、スマホ表示
  </Card>
  <Card title="大事な前提" icon="information">
    - Windows 64bit のPCで動きます。
    - Ollama と AIモデルは、初回セットアップで自動的に用意します（インターネットが必要）。
    - 以降の生成は、あなたのPCの中だけで行われます。
  </Card>
</CardGrid>
