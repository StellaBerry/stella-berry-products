**Role Garden WebUI は、あなたのPCの中だけで動きます。入力も生成物も外に出ません。**
要点は次の3つです。

<CardGrid>
  <Card title="通信しない" icon="laptop">
    外部と通信する機能はありません（AIモデルのダウンロードを除く）。
  </Card>
  <Card title="学習しない" icon="seti:default">
    あなたの入力でAIモデルが書き換わることはありません。
  </Card>
  <Card title="記憶はローカル" icon="open-book">
    キャラの記憶は「日記」としてPC内に保存され、外へ出ません。
  </Card>
</CardGrid>

## 「学習」と「記憶」は別物です

「AIに入力する」と「AIが学習する」は、よく混同されますが別のことです。

<CardGrid>
  <Card title="学習：しません" icon="close">
    土台となるAIモデルは完成済みで、あなたが使っても中身が書き換わることはありません。あなたの入力が、他の人の環境や元のAIに反映されることもありません。
  </Card>
  <Card title="記憶：します（PC内の日記）" icon="approve-check">
    キャラクターがあなたを覚えていくのは、会話の要点を日記としてPC内に保存し、生成のたびに読み返しているためです。この日記も外へは出ません。
  </Card>
</CardGrid>

:::note
クラウド型のAIサービスでは、入力が学習に使われる場合もあります。一方、本アプリのようにPC内で動くローカルAIは学習を行いません。
:::

## 例外

- 外部と通信するのは **初回のモデルのダウンロード時のみ** です。以降の生成では通信しません。
- ユーザーご自身の操作（SNS投稿、生成文のコピー、クラウド同期など）でデータを外へ出すのは、アプリの機能とは別の話です。
- **LAN公開** を任意でONにした場合のみ、同じ家庭内ネットワークにある自分の端末からアクセスできます。これはインターネットや第三者へ送るものではありません（[スマホ接続](/stella-berry-products/role-garden-webui/mobile/)）。

## 個人利用と公開について

Role Garden WebUI には、外部と通信する機能がありません（AIモデルのダウンロードを除く）。入力・会話・生成物がインターネットへ送られることはなく、作者が中身を見ることもできません。

また、あなたの入力がAIの学習に使われることもありません。土台となるAIモデルは完成済みで、使っても中身が書き換わることはなく、あなたの入力が他の人の環境や元のAIに反映されることもありません。

キャラクターがあなたの好みや話したことを"覚えて"いくのは、アプリが会話を「日記」のようにあなたのPC内へ書き留め、メッセージを生成するときに毎回読み返しているためです。この日記もあなたのPCの中だけにあり、外へ出ることはありません。

そのため、ご自身が権利を持たないキャラクターを設定して"なりきり"を楽しむこともできます。すべてPC内・あなただけで楽しむ限り、「個人利用」の範囲に収まる設計です。

公開・共有を制限することはありません。ただし、権利を持たないキャラクターを扱った画面や生成物をSNS等へ公開すると、権利者などとのトラブルの原因になり得ます。個人利用の範囲にとどめることをおすすめしますが、公開するかどうかはご自身の判断と責任でお決めください。制作者は、その結果生じた第三者間のトラブルについて一切関与せず、責任を負いません。
