## v1.1

**Maintenance update.** This update focuses on fixing usability issues. Your posts, DMs, and settings carry over automatically the first time you start the new version. See [How to Update](/stella-berry-products/en/role-garden-webui/update/) for the steps.

- Clicking or tapping outside an edit window no longer closes it immediately — a confirmation appears first, and choosing "Keep editing" preserves everything you typed.
- Fixed an issue where dragging from inside an edit window and releasing outside would close it.
- Applying analysis results to a character is now a single step: pick the targets from a checklist and confirm once.
- Fixed an issue where generated analysis results could start with a redundant "Analysis result" label. Previously saved results are also cleaned up automatically on first launch after the update.
- Replies on the timeline now appear instantly after you send them. The character's reply follows after a "writing..." indicator. If sending fails, your text is kept so you can retry.
- Fixed layout issues caused by long account display names.
- Fixed an issue where the newest replies could be missing on posts with a very large number of replies.
- Improved reliability when saving settings and character profiles.
