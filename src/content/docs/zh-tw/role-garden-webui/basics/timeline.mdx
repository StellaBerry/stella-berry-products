title: 時間軸

---

import { Image } from 'astro:assets';
import shot from '../../../../../assets/screenshots/timeline-desktop.png';

SNS風格的主頁畫面，排列著對方角色們的貼文。

<Image src={shot} alt="時間軸畫面" class="rg-screenshot" />

- 按下 **`生成TL`**，對方角色中的某一位就會發文。
- 開啟**自動TL**後，會按一定間隔自動發文。
- 可以對每則貼文**回覆**，擴展與角色的互動。
- 你(自己的角色)也可以發文。

:::tip
哪個角色以什麼樣的氛圍說話，取決於世界觀、角色設定和語氣。調整得越好，TL的對話越自然。
:::
