適用於 Windows 64bit。無需複雜設定，**解壓縮zip、雙擊啟動器即可**開始。

## 安裝步驟

<Steps>

1. 將發行版zip解壓縮到任意位置。
2. 雙擊 **`Role Garden WebUI Launcher.exe`**。
3. 之後啟動器會為你的電腦準備合適的模型並直接啟動。

</Steps>

首次執行時，只需按照畫面指引進行：確認電腦規格、選擇儲存位置、下載模型(附進度條)。你的操作基本上只有「雙擊」和「確認模型」。

## 需要的東西

- Windows 64bit 電腦
- 首次設定時的網路連線

## 設定時自動準備的東西

- Ollama(如果電腦上沒有)
- 適合你電腦的AI模型

## 模型的選擇

啟動器會**自動推薦**適合你GPU的模型。拿不準的話，按推薦進行即可。如果試圖選擇過重的模型，會跳出警告。

| 推薦 | 模型 | 參考 |
| --- | --- | --- |
| `low`  | gemma4:e2b, gemma4:e4b | 較小的GPU / 以CPU為主 |
| `mid`  | + qwen3:14b | VRAM 11GB以上 |
| `high` | + gemma4:26b | VRAM 20GB以上 |

:::caution
過重的模型可能導致生成非常耗時，或看起來像當機。拿不準的話，請先從推薦開始。
:::

## 儲存位置

資料和模型的儲存位置可以在首次執行的畫面中選擇。

保持預設也沒有問題。

## 啟動·停止

- **啟動**: `Role Garden WebUI Launcher.exe`
- **停止**: 啟動器的停止按鈕
