새 버전은 별도의 폴더로 압축이 풀리므로 덮어쓰기 작업은 없습니다. 만든 캐릭터·게시물·설정은 앱과 다른 위치에 저장되어 있어 **새 버전을 실행하기만 하면 자동으로 이어집니다**.

## 업데이트 순서

<Steps>

1. Role Garden WebUI를 종료합니다(런처의 중지 버튼 또는 `stop_role_garden_webui.bat`).

2. 구입한 스토어(BOOTH / Gumroad)에서 새 zip을 내려받아 원하는 위치에 압축을 풉니다(압축을 풀면 `RoleGardenWebUI_v1_1` 같은 폴더가 생깁니다).

3. 그 안의 `RoleGardenWebUI` 폴더를 열고 "Role Garden WebUI Launcher.exe"를 실행합니다. 평소처럼 사용할 수 있는지 확인하세요. 재설정은 필요 없습니다.

4. 바탕 화면에 바로 가기를 두고 있다면 다시 만듭니다(이전 바로 가기는 이전 버전의 폴더를 가리키고 있기 때문입니다). 이전 바로 가기를 삭제하고, 새 "Role Garden WebUI Launcher.exe"를 마우스 오른쪽 클릭 → "보내기" → "바탕 화면에 바로 가기 만들기"를 선택하세요.

5. 새 버전의 동작을 확인했다면 이전 버전의 폴더는 통째로 삭제해도 됩니다.

</Steps>

## 참고

- 내려받은 AI 모델과 Ollama는 그대로 사용됩니다. 다시 내려받을 필요가 없습니다.
- 스마트폰에서 이용(LAN 공개)하는 경우, 업데이트 후 첫 실행 시 Windows 방화벽 허용 화면이 다시 표시될 수 있습니다. 표시되면 "액세스 허용"을 선택하세요.
- (고급) `models_custom.txt`를 직접 만들어 사용한 경우에만 새 폴더로 복사하세요.
