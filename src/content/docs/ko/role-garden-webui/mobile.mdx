초기 상태에서는 실행 중인 PC에서만 사용할 수 있습니다. 같은 Wi-Fi의 스마트폰이나 태블릿에서 사용하려면, 설정에서 **LAN 공개**를 켜고 서버를 재시작하세요.

## 공개 모드

| 설정 | 설명 |
| --- | --- |
| PC만 | 가장 안전합니다. 실행 중인 PC에서만 사용할 수 있습니다. |
| 공개(PIN 인증 있음) | 같은 LAN의 스마트폰 등에서 접속할 수 있습니다. 런처에 표시된 PIN 입력이 필요합니다. |
| 공개(PIN 인증 없음) | PIN 입력 없이 접속할 수 있습니다. 집처럼 신뢰할 수 있는 네트워크에서만 사용하세요. |

## 연결 순서

<Steps>

1. 설정에서 LAN 공개를 선택합니다.
2. 재시작 안내가 표시되면 서버를 재시작합니다.
3. 런처의 모바일용 URL 또는 QR 코드를 스마트폰에서 엽니다.
4. PIN 인증이 있는 경우, 런처에 표시된 PIN을 입력합니다.

</Steps>

:::caution
LAN 공개는 가정 내 LAN용 기능입니다. 인터넷에 직접 공개하지 마세요.
모바일용 URL은 `http://`로 동작하므로 통신이 암호화되지 않습니다. 공용 Wi-Fi나 신뢰할 수 없는 LAN에서는 사용하지 말고, 집처럼 신뢰할 수 있는 네트워크에서만 사용하세요.
:::
