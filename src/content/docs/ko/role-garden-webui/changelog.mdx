Role Garden WebUI의 중요한 공지, 업데이트 정보, 이용약관 변경 공지를 게재합니다.

## v1.1

**수정 업데이트.** 조작 관련 문제를 중심으로 수정했습니다. 업데이트 후에도 기존 게시물·DM·설정은 그대로 유지됩니다(새 버전 첫 실행 시 자동으로 이어집니다). 순서는 [업데이트 방법](/stella-berry-products/ko/role-garden-webui/update/)을 참고하세요.

- 편집 화면 바깥을 실수로 클릭/탭해도 바로 닫히지 않고 확인이 표시됩니다. "계속 편집"을 선택하면 입력 중인 내용이 그대로 남습니다.
- 편집 화면 안에서 드래그하다가 바깥에서 놓으면 화면이 닫히는 문제를 수정했습니다.
- 분석 결과를 캐릭터에 반영하는 작업이 체크리스트에서 선택 후 한 번의 확인으로 완료되도록 개선되었습니다.
- 분석 결과 첫머리에 "분석 결과"라는 불필요한 머리말이 섞이는 문제를 수정했습니다. 이전에 저장된 분석 결과도 업데이트 후 첫 실행 시 자동으로 정리됩니다.
- 타임라인에서 답글을 보내면 내 답글이 즉시 표시됩니다. 상대의 답글은 "작성 중..." 표시 후에 도착합니다. 전송에 실패해도 입력한 내용은 사라지지 않고 다시 보낼 수 있습니다.
- 계정 표시 이름이 길 때 화면 레이아웃이 깨지는 문제를 수정했습니다.
- 답글이 매우 많은 게시물에서 최신 답글이 표시되지 않는 문제를 수정했습니다.
- 설정과 캐릭터 정보를 저장할 때의 안정성을 개선했습니다.
