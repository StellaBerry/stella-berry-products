Windows 64bit용입니다. 어려운 설정은 필요 없습니다. **zip을 풀고 런처를 더블클릭하기만 하면** 시작할 수 있습니다.

## 설치 순서

<Steps>

1. 배포 zip을 원하는 위치에 풉니다.
2. **`Role Garden WebUI Launcher.exe`**를 더블클릭합니다.
3. 그 다음은 런처가 PC에 맞는 모델을 준비하고 그대로 실행합니다.

</Steps>

첫 실행 시에는 PC 사양 확인·저장 위치 선택·모델 다운로드(진행 바 표시)까지 화면 안내를 따라가기만 하면 됩니다. 여러분의 조작은 기본적으로 「더블클릭」과 「모델 확인」뿐입니다.

## 필요한 것

- Windows 64bit PC
- 첫 설정 시 인터넷 연결

## 설정에서 자동으로 준비되는 것

- Ollama(PC에 없는 경우)
- PC에 맞는 AI 모델

## 모델 고르는 법

런처가 GPU에 맞는 모델을 **자동으로 추천**합니다. 잘 모르겠다면 추천 그대로 진행하면 됩니다. 너무 무거운 모델을 고르려 하면 경고가 표시됩니다.

| 추천 | 모델 | 기준 |
| --- | --- | --- |
| `low`  | gemma4:e2b, gemma4:e4b | 작은 GPU / CPU 중심 |
| `mid`  | + qwen3:14b | VRAM 11GB 이상 |
| `high` | + gemma4:26b | VRAM 20GB 이상 |

:::caution
너무 무거운 모델은 생성에 매우 오래 걸리거나 멈춘 것처럼 보일 수 있습니다. 잘 모르겠다면 추천부터 시작하세요.
:::

## 저장 위치

데이터와 모델의 저장 위치는 첫 화면에서 고를 수 있습니다.

기본값 그대로도 문제없습니다.

## 시작·정지

- **시작**: `Role Garden WebUI Launcher.exe`
- **정지**: 런처의 정지 버튼
