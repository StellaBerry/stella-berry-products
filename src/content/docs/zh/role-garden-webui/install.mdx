适用于 Windows 64bit。无需复杂设置，**解压zip、双击启动器即可**开始。

## 安装步骤

<Steps>

1. 将发行版zip解压到任意位置。
2. 双击 **`Role Garden WebUI Launcher.exe`**。
3. 之后启动器会为你的电脑准备合适的模型并直接启动。

</Steps>

首次运行时，只需按照画面指引进行：确认电脑规格、选择保存位置、下载模型(带进度条)。你的操作基本上只有「双击」和「确认模型」。

## 需要的东西

- Windows 64bit 电脑
- 首次设置时的网络连接

## 设置时自动准备的东西

- Ollama(如果电脑上没有)
- 适合你电脑的AI模型

## 模型的选择

启动器会**自动推荐**适合你GPU的模型。拿不准的话，按推荐进行即可。如果试图选择过重的模型，会弹出警告。

| 推荐 | 模型 | 参考 |
| --- | --- | --- |
| `low`  | gemma4:e2b, gemma4:e4b | 较小的GPU / 以CPU为主 |
| `mid`  | + qwen3:14b | VRAM 11GB以上 |
| `high` | + gemma4:26b | VRAM 20GB以上 |

:::caution
过重的模型可能导致生成非常耗时，或看起来像卡住了。拿不准的话，请先从推荐开始。
:::

## 保存位置

数据和模型的保存位置可以在首次运行的画面中选择。

保持默认也没有问题。

## 启动·停止

- **启动**: `Role Garden WebUI Launcher.exe`
- **停止**: 启动器的停止按钮

## 卸载

双击 `Role Garden WebUI Uninstaller.exe`，勾选想要删除的项目并执行。

- 可选项目: 快捷方式 / Ollama / AI模型 / 用户数据(均为可选)。
- Ollama 和AI模型可能被其他工具使用，是否删除由你自由选择。
- 应用本体只需删除此文件夹即可(执行结束时也会显示提示)。
