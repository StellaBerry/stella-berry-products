import { Image } from 'astro:assets';

SNS风格的主页画面，排列着对方角色们的帖子。

<Image src={shot} alt="时间线画面" class="rg-screenshot" />

- 按下 **`生成TL`**，对方角色中的某一位就会发帖。
- 开启**自动TL**后，会按一定间隔自动发帖。
- 可以对每条帖子**回复**，扩展与角色的互动。
- 你(自己的角色)也可以发帖。

:::tip
哪个角色以什么样的氛围说话，取决于世界观、角色设定和语气。调整得越好，TL的对话越自然。
:::
