Overview

Role Garden WebUI is a local creative tool for playing with characters in a social-network-like interface. You create your own account-like character, create worlds and partner characters, then interact through timelines, replies, DMs, and roleplay-style text.

Main features

  • World, player character, and partner character setup
  • Timeline posts, replies, and direct messages
  • Roleplay-friendly replies with actions and scene description
  • Writing-style analysis from sample text
  • Themes, color palettes, and mobile layout

Before you start

  • Designed for Windows 64-bit.
  • Ollama and LLM models are not bundled.
  • Initial setup and model downloads require internet access.
  • After models are downloaded, normal generation runs on your PC through local LLMs.

LAN sharing is intended for trusted home networks. Do not expose the WebUI directly to the internet.

Installation

If you are unsure where to begin, start with the low-spec setup. Large models may make your PC slow, take a long time to respond, or freeze on some systems.

  1. Extract the distributed zip file.
  2. For a lightweight first test, run install_low_spec.bat.
  3. For a menu-based setup, run install_role_garden_webui.bat.
  4. After installation, open Role Garden WebUI Launcher.exe.
  5. The launcher starts the local server and opens the WebUI in your browser.
MenuPurpose

Basic use

First setup

  1. Create a world.
  2. Create your own character.
  3. Create partner characters.

Worlds

A world defines the setting and atmosphere: fantasy, modern school, near future, TRPG-like scenarios, or anything you want to play in.

Your character

Your character works like your account in the social UI. You can set names, profile text, icons, and header images.

Partner characters

Partner characters participate in timeline posts, replies, and DMs. Profiles, tone, likes, and relationship notes help guide generation.

Timeline

Press Generate TL to add a post from one of the partner characters. Auto TL can post at intervals.

DM and roleplay

DMs are one-on-one conversations. Text can include dialogue, movement, expressions, distance, gaze, and scene descriptions.

Analysis

The analysis tab can analyze writing samples and help shape a character's speech style or role text.

Appearance

You can switch between flat and neumorphism styles, themes, color palettes, and UI languages.

Mobile access and LAN sharing

By default, Role Garden WebUI is available only on the PC that runs it. To use it from a phone or tablet on the same Wi-Fi, enable LAN sharing in settings and restart the server.

ModeDescription
PC onlySafest mode. Only the host PC can access the WebUI.
Public with PINDevices on the same LAN can connect after entering the PIN shown in the launcher.
Public without PINDevices on the same LAN can connect without a PIN. Use only on trusted home networks.

The mobile URL uses http://, so traffic is not encrypted. Do not use it on public or untrusted Wi-Fi. A connected device has full management access, including settings and deletion actions.

Troubleshooting

Cannot start

  • Extract the zip before running files.
  • Run Role Garden WebUI Launcher.exe or the startup bat file.
  • Check whether security software blocked Python, bat, or exe files.

Ollama

  • Install Ollama from the official site if needed.
  • Existing Ollama installations are reused when possible.
  • Model downloads require time and storage.

Slow generation

  • You may be using a model that is too heavy for your PC.
  • Test with a lightweight model first.
  • Speed depends on RAM, VRAM, GPU, CPU, model, and Ollama state.

Phone cannot connect

  • Check that PC and phone are on the same Wi-Fi.
  • Allow Python or port 7880 through Windows Firewall.
  • Check router client isolation, VPNs, and virtual networks.

Typical data folder: Documents\RoleGardenWebUI\. Back it up before updates, reinstalling, uninstalling, resetting data, or manual edits.
