简介

Role Garden WebUI 是一款用于角色创作和角色扮演的本地 WebUI。你可以像创建 SNS 账号一样创建自己的角色，再创建世界观和对话对象角色，通过时间线、回复和 DM 进行互动。

主要功能

  • 创建世界观、自己的角色、对方角色
  • SNS 风时间线、回复、DM
  • 适合角色扮演的回复生成
  • 文风与语气分析
  • 主题、配色、手机布局

使用前须知

  • 面向 Windows 64bit。
  • 不附带 Ollama 本体或 LLM 模型。
  • 初次安装和下载模型需要网络连接。
  • 下载模型后，通常生成会通过本地 LLM 在你的 PC 上运行。

LAN 公开功能仅适合可信任的家庭网络。请不要直接公开到互联网。

安装

如果不确定从哪里开始，建议先使用轻量配置。过大的模型可能让 PC 变得很慢、生成耗时很久，或在部分环境中卡住。

  1. 解压发布的 zip 文件。
  2. 如需先用轻量配置测试，请运行 install_low_spec.bat。
  3. 如需通过菜单选择安装内容，请运行 install_role_garden_webui.bat。
  4. 安装后打开 Role Garden WebUI Launcher.exe。
  5. 启动器会启动本地服务器，并在浏览器中打开 WebUI。
菜单用途
low用于首次测试的轻量配置。
mid包含中等规模模型，适合在轻量与质量之间取得平衡。
high高配置 PC 用。仅在存储、RAM、VRAM 充足时选择。
env只准备 Python、WebUI 专用环境和 Ollama，不下载模型。

基本操作

首先创建

  1. 世界观
  2. 自己的角色
  3. 对方角色

世界观

世界观定义角色所在的地点和气氛。可以是幻想、现代、校园、近未来、TRPG 剧本风等。

自己的角色

自己的角色像是你在 SNS 里的账号。可以设置名称、显示名、简介、头像、头图等。

对方角色

对方角色会参与 TL 投稿、回复和 DM 回复。简介、语气、喜好、关系备忘会影响生成。

时间线

点击 生成 TL 后，对方角色中的某一位会发布内容。开启自动 TL 后，会按一定间隔发布。

DM 与角色扮演

DM 可进行一对一对话。文本可包含台词、动作、表情、距离、视线和场景描写。

分析

分析页面可从样本文本中分析文风和语气，用于调整角色说话方式。

外观

可切换扁平/拟物柔和风格、主题、配色和 UI 语言。

手机连接与 LAN 公开

默认情况下，只有运行 WebUI 的 PC 可以访问。若想从同一 Wi-Fi 内的手机或平板使用，请在设置中启用 LAN 公开并重启服务器。

设置说明
仅 PC最安全。只有运行中的 PC 可以访问。
公开（需要 PIN）同一 LAN 内设备输入启动器显示的 PIN 后可连接。
公开（无需 PIN）同一 LAN 内设备无需 PIN 即可连接。仅在可信家庭网络中使用。

手机 URL 使用 http://，通信不会加密。请勿在公共 Wi-Fi 或不可信 LAN 中使用。连接后的设备拥有完整管理权限，包括设置变更和删除操作。

故障排查

无法启动

  • 请先解压 zip 再运行。
  • 运行 Role Garden WebUI Launcher.exe 或启动 bat。
  • 确认安全软件没有阻止 Python、bat 或 exe。

Ollama

  • 如未安装 Ollama，请从官方网站安装。
  • 已有 Ollama 时会尽量复用。
  • 下载模型需要时间和存储空间。

生成很慢

  • 可能选择了对 PC 来说过重的模型。
  • 请先用轻量模型测试。
  • 速度取决于 RAM、VRAM、GPU、CPU、模型和 Ollama 状态。

手机打不开

  • 确认 PC 和手机连接同一 Wi-Fi。
  • 在 Windows 防火墙中允许 Python 或端口 7880。
  • 检查路由器客户端隔离、VPN、虚拟网络等影响。

通常数据目录为 Documents\RoleGardenWebUI\。更新、重装、卸载、数据初始化或手动编辑前，请按需备份。
