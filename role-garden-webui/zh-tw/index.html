Role Garden WebUI

這是一款 Windows 用的本機 WebUI，可透過 Ollama 與本機 LLM 進行角色創作、SNS 風時間軸、回覆、DM 與角色扮演。

簡介

Role Garden WebUI 是一款用於角色創作與角色扮演的本機 WebUI。你可以像建立 SNS 帳號一樣建立自己的角色，再建立世界觀與對話角色，透過時間軸、回覆與 DM 互動。

主要功能

  • 建立世界觀、自己的角色、對方角色
  • SNS 風時間軸、回覆、DM
  • 適合角色扮演的回覆生成
  • 文風與語氣分析
  • 主題、色盤、手機版面

使用前須知

  • 面向 Windows 64bit。
  • 不包含 Ollama 本體或 LLM 模型。
  • 初次設定與下載模型需要網路連線。
  • 模型下載後，一般生成會透過本機 LLM 在你的 PC 上執行。

LAN 公開功能僅適合可信任的家用網路。請不要直接公開到網際網路。

安裝

如果不確定從哪裡開始，建議先使用輕量設定。過大的模型可能讓 PC 變得很慢、生成耗時很久，或在部分環境中卡住。

  1. 解壓縮發布的 zip 檔。
  2. 若要先用輕量設定測試，請執行 install_low_spec.bat。
  3. 若要透過選單選擇安裝內容，請執行 install_role_garden_webui.bat。
  4. 安裝後開啟 Role Garden WebUI Launcher.exe。
  5. 啟動器會啟動本機伺服器，並在瀏覽器中開啟 WebUI。
選單用途
low用於首次測試的輕量設定。
mid包含中等規模模型，適合在輕量與品質之間取得平衡。
high高階 PC 用。僅在儲存空間、RAM、VRAM 充足時選擇。
env只準備 Python、WebUI 專用環境與 Ollama，不下載模型。

基本操作

首先建立

  1. 世界觀
  2. 自己的角色
  3. 對方角色

世界觀

世界觀定義角色所在的地點與氣氛。可以是奇幻、現代、校園、近未來、TRPG 劇本風等。

自己的角色

自己的角色就像你在 SNS 裡的帳號。可以設定名稱、顯示名稱、簡介、頭像、頁首圖片等。

對方角色

對方角色會參與 TL 投稿、回覆與 DM 回覆。簡介、語氣、喜好、關係備忘會影響生成。

時間軸

按下 生成 TL 後，某位對方角色會發布內容。開啟自動 TL 後，會依一定間隔發布。

DM 與角色扮演

DM 可進行一對一對話。文字可包含台詞、動作、表情、距離、視線與場景描寫。

分析

分析頁面可從樣本文本中分析文風與語氣，用於調整角色說話方式。

外觀

可切換扁平/柔和擬物風格、主題、色盤與 UI 語言。

手機連線與 LAN 公開

預設情況下，只有執行 WebUI 的 PC 可以存取。若想從同一 Wi-Fi 內的手機或平板使用，請在設定中啟用 LAN 公開並重新啟動伺服器。

設定說明
僅 PC最安全。只有執行中的 PC 可以存取。
公開（需要 PIN）同一 LAN 內裝置輸入啟動器顯示的 PIN 後可連線。
公開（無需 PIN）同一 LAN 內裝置不需 PIN 即可連線。僅在可信任的家用網路中使用。

手機 URL 使用 http://，通訊不會加密。請勿在公共 Wi-Fi 或不可信任的 LAN 中使用。連線後的裝置具有完整管理權限，包括設定變更與刪除操作。

疑難排解

無法啟動

  • 請先解壓縮 zip 再執行。
  • 執行 Role Garden WebUI Launcher.exe 或啟動 bat。
  • 確認安全軟體沒有阻擋 Python、bat 或 exe。

Ollama

  • 如未安裝 Ollama，請從官方網站安裝。
  • 已有 Ollama 時會盡量重複使用。
  • 下載模型需要時間與儲存空間。

生成很慢

  • 可能選擇了對 PC 來說過重的模型。
  • 請先用輕量模型測試。
  • 速度取決於 RAM、VRAM、GPU、CPU、模型與 Ollama 狀態。

手機打不開

  • 確認 PC 與手機連接同一 Wi-Fi。
  • 在 Windows 防火牆中允許 Python 或連接埠 7880。
  • 檢查路由器用戶端隔離、VPN、虛擬網路等影響。

一般資料夾為 Documents\RoleGardenWebUI\。更新、重裝、解除安裝、資料初始化或手動編輯前，請視需要備份。
