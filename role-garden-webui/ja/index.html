はじめに

Role Garden WebUIは、キャラクター創作やロールプレイをSNS風の画面で遊ぶためのローカルWebUIです。自分側キャラクターをSNSアカウントのように作成し、そのキャラクターに紐づく相手キャラクターたちと、タイムラインやDMでやり取りできます。

できること

  • 世界観、自キャラ、相手キャラの作成
  • SNS風タイムライン、リプライ、DM
  • ロールプレイ向け返信生成
  • 文体・口調分析
  • テーマ、カラーパレット、スマホ表示

大事な前提

  • Windows 64bit向けです。
  • Ollama本体やLLMモデルは同梱していません。
  • 初回セットアップとモデル取得にはインターネット接続が必要です。
  • モデル取得後の通常生成はPC内のローカルLLMで行われます。

LAN公開は家庭内LAN向けです。インターネットへ直接公開しないでください。

インストール

最初に迷った場合は、軽量構成から始めてください。重いモデルを無理に使うと、PCが非常に重くなったり、生成に時間がかかったり、環境によってはフリーズする場合があります。

  1. 配布zipを展開します。
  2. まず軽量構成で試す場合は install_low_spec.bat を実行します。
  3. メニュー式で選びたい場合は install_role_garden_webui.bat を実行します。
  4. インストール後、Role Garden WebUI Launcher.exe を開きます。
  5. ランチャーが起動し、しばらくするとブラウザでWebUIが開きます。
メニュー用途
lowまず動作確認したい方向けの軽量構成です。迷った場合はここから始めてください。
mid軽量構成に加えて中規模モデルを取得します。軽さと品質のバランスを試したい方向けです。
high高スペックPC向けです。容量、RAM、VRAMに余裕がある場合のみ選んでください。
envPython、WebUI専用環境、Ollamaだけを準備します。モデルは取得しません。

インストールされるもの

  • Python 3.11 64bit、または既存のPython 3.10以上 64bit版
  • このフォルダ内のWebUI専用Python環境
  • Ollama
  • 選択した構成に応じたローカルLLMモデル

既に利用可能なPythonやOllama、取得済みモデルがある場合は、できるだけ既存のものを再利用します。

基本操作

最初に作るもの

  1. 世界観を作る
  2. 自キャラを作る
  3. 相手キャラを作る

世界観

自キャラがいる場所や空気を定義します。ファンタジー、現代、学園、近未来、TRPGシナリオ風など、遊びたい世界の雰囲気を書いてください。

自キャラ

自分が操作するSNSアカウントのようなものです。名前、表示名、プロフィール、アイコン、ヘッダー画像などを設定できます。

相手キャラ

TL投稿、リプライ、DM返信に参加するAIキャラクターです。プロフィール、口調、好きなもの、関係性メモなどを設定できます。

タイムライン

TLを生成 を押すと、相手キャラクターの誰かが投稿します。自動TLをONにすると一定間隔で投稿されます。

DMとロールプレイ

DMでは一対一で会話できます。セリフだけでなく、動き、表情、距離、視線、場面描写などを含む文章にも対応しています。

分析

分析タブでは、サンプル文から文体や口調の分析を行えます。キャラクターの話し方やロール内の書き方を整えるために使えます。

見た目

フラット/ニューモーフィズム表示、テーマ、カラーパレット、言語表示を切り替えられます。

スマホ接続とLAN公開

初期状態では、起動しているPCからだけ使えます。同じWi-Fi内のスマホやタブレットから使いたい場合は、設定でLAN公開を有効にし、サーバーを再起動してください。

設定説明
PCのみ一番安全です。起動しているPCからだけ使えます。
公開（PIN認証あり）同じLAN内のスマホや別端末から接続できます。ランチャーに表示されたPINの入力が必要です。
公開（PIN認証なし）PIN入力なしで接続できます。自宅など信頼できるネットワーク内でだけ使ってください。
  1. 設定でLAN公開を選びます。
  2. 設定欄に再起動案内が出たら、サーバーを再起動します。
  3. ランチャーのスマホ用URLまたはQRコードをスマホで開きます。
  4. PIN認証ありの場合は、ランチャーに表示されたPINを入力します。

スマホ用URLは http:// で動作するため通信内容は暗号化されません。共有Wi-Fiや信用できないLANでは使わず、自宅など信頼できるネットワーク内だけで使ってください。

トラブル対応

起動しない

  • zipを展開してから実行してください。
  • Role Garden WebUI Launcher.exe または起動batを実行してください。
  • セキュリティソフトがPython、bat、exeを止めていないか確認してください。

Ollama関連

  • Ollamaが未導入の場合は公式サイトから導入してください。
  • 既存Ollamaがある場合は再利用します。
  • モデル取得には時間とストレージ容量が必要です。

生成が遅い

  • 重いモデルを選んでいる可能性があります。
  • まず軽量モデルで動作確認してください。
  • 速度はPC性能、モデル、Ollamaの状態に左右されます。

スマホから開けない

  • PCとスマホが同じWi-Fiか確認してください。
  • WindowsファイアウォールでPythonまたはポート7880が許可されているか確認してください。
  • ルーターの端末分離、VPN、仮想ネットワークの影響も確認してください。

バックアップ

通常のデータ保存先は Documents\RoleGardenWebUI\ です。アップデート、再インストール、アンインストール、データ初期化、手動編集の前には必要に応じてバックアップしてください。
