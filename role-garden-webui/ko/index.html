소개

Role Garden WebUI는 캐릭터 창작과 롤플레이를 SNS 같은 화면에서 즐기기 위한 로컬 WebUI입니다. 자신의 캐릭터를 계정처럼 만들고, 세계관과 상대 캐릭터를 설정한 뒤 타임라인, 답글, DM으로 상호작용할 수 있습니다.

주요 기능

  • 세계관, 내 캐릭터, 상대 캐릭터 작성
  • SNS풍 타임라인, 답글, DM
  • 행동과 장면 묘사를 포함한 롤플레이 응답
  • 문체와 말투 분석
  • 테마, 컬러 팔레트, 모바일 레이아웃

중요한 전제

  • Windows 64bit용입니다.
  • Ollama 본체와 LLM 모델은 포함되어 있지 않습니다.
  • 초기 설정과 모델 다운로드에는 인터넷 연결이 필요합니다.
  • 모델 다운로드 후 일반 생성은 PC 안의 로컬 LLM에서 실행됩니다.

LAN 공개는 신뢰할 수 있는 가정 내 네트워크용 기능입니다. 인터넷에 직접 공개하지 마세요.

설치

처음이라면 가벼운 구성부터 시작하는 것을 권장합니다. 무거운 모델은 PC를 매우 느리게 만들거나, 응답이 오래 걸리거나, 환경에 따라 멈출 수 있습니다.

  1. 배포 zip 파일을 압축 해제합니다.
  2. 가벼운 구성으로 먼저 시험하려면 install_low_spec.bat을 실행합니다.
  3. 메뉴에서 선택하려면 install_role_garden_webui.bat을 실행합니다.
  4. 설치 후 Role Garden WebUI Launcher.exe를 엽니다.
  5. 런처가 로컬 서버를 시작하고 브라우저에서 WebUI를 엽니다.
메뉴용도
low처음 동작 확인을 위한 경량 구성입니다.
mid중간 규모 모델을 포함한 균형형 구성입니다.
high저장공간, RAM, VRAM에 여유가 있는 고사양 PC용입니다.
envPython, WebUI 전용 환경, Ollama만 준비하고 모델은 받지 않습니다.

기본 사용

처음 만들 것

  1. 세계관
  2. 내 캐릭터
  3. 상대 캐릭터

세계관

캐릭터가 있는 장소와 분위기를 정합니다. 판타지, 현대, 학원, 근미래, TRPG 시나리오풍 등 원하는 설정을 적어 주세요.

내 캐릭터

사용자가 조작하는 SNS 계정 같은 캐릭터입니다. 이름, 표시명, 프로필, 아이콘, 헤더 이미지를 설정할 수 있습니다.

상대 캐릭터

TL 게시글, 답글, DM 응답에 참여하는 AI 캐릭터입니다. 프로필, 말투, 좋아하는 것, 관계 메모가 생성에 영향을 줍니다.

타임라인

TL 생성을 누르면 상대 캐릭터 중 누군가가 글을 올립니다. 자동 TL을 켜면 일정 간격으로 글이 올라옵니다.

DM과 롤플레이

DM에서는 일대일 대화를 할 수 있습니다. 대사뿐 아니라 행동, 표정, 거리, 시선, 장면 묘사도 사용할 수 있습니다.

분석

분석 탭에서는 샘플 문장에서 문체와 말투를 분석할 수 있습니다.

외형

플랫/뉴모피즘 표시, 테마, 컬러 팔레트, UI 언어를 전환할 수 있습니다.

스마트폰 접속과 LAN 공개

기본 상태에서는 실행 중인 PC에서만 사용할 수 있습니다. 같은 Wi-Fi의 스마트폰이나 태블릿에서 사용하려면 설정에서 LAN 공개를 켜고 서버를 다시 시작하세요.

설정설명
PC만가장 안전한 설정입니다. 실행 중인 PC에서만 접속할 수 있습니다.
공개(PIN 인증 있음)같은 LAN의 기기에서 런처에 표시된 PIN을 입력해 접속합니다.
공개(PIN 인증 없음)PIN 없이 접속합니다. 신뢰할 수 있는 가정 내 네트워크에서만 사용하세요.

스마트폰용 URL은 http://로 동작하며 암호화되지 않습니다. 공용 Wi-Fi나 신뢰할 수 없는 LAN에서는 사용하지 마세요. 접속한 기기는 설정 변경과 삭제를 포함한 관리 권한을 가집니다.

문제 해결

실행되지 않음

  • zip을 압축 해제한 뒤 실행하세요.
  • Role Garden WebUI Launcher.exe 또는 시작 bat 파일을 실행하세요.
  • 보안 소프트웨어가 Python, bat, exe를 차단하지 않았는지 확인하세요.

Ollama

  • 필요하면 공식 사이트에서 Ollama를 설치하세요.
  • 기존 Ollama는 가능한 한 재사용합니다.
  • 모델 다운로드에는 시간과 저장공간이 필요합니다.

생성이 느림

  • PC에 비해 모델이 너무 무거울 수 있습니다.
  • 먼저 가벼운 모델로 확인하세요.
  • 속도는 RAM, VRAM, GPU, CPU, 모델, Ollama 상태에 따라 달라집니다.

스마트폰에서 열리지 않음

  • PC와 스마트폰이 같은 Wi-Fi인지 확인하세요.
  • Windows 방화벽에서 Python 또는 포트 7880을 허용하세요.
  • 라우터의 기기 분리, VPN, 가상 네트워크를 확인하세요.

일반 데이터 폴더는 Documents\RoleGardenWebUI\입니다. 업데이트, 재설치, 제거, 데이터 초기화, 수동 편집 전 필요한 데이터를 백업하세요.
