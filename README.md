Public GitHub Pages source for product manuals and support pages.

- `role-garden-webui/` - Role Garden WebUI support and manuals

Branch policy:

- `main` is the published GitHub Pages source.
- `develop` is the working branch for draft updates.
- Merge `develop` into `main` only when the public site should be updated.
